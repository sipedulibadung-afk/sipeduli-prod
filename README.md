- ✅ Backend Google Sheets (gratis)

## 🚀 Instalasi

### Frontend
1. Upload semua file ke GitHub Pages / hosting

### Backend
1. Buat Google Spreadsheet baru
2. Buat folder di Google Drive untuk upload file
3. Buka script.google.com → New Project
4. Salin Code.gs, ganti SPREADSHEET_ID dan FOLDER_ID
5. Jalankan fungsi `setupDatabase()`
6. Deploy sebagai Web App (akses: Anyone)
7. Salin URL ke `assets/js/api.js` → `BASE_URL`
